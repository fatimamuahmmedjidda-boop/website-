# Stayza Real Estate Website

## Files Included:
- index.html (Home page)
- about.html (About, Mission, Vision, CEO)
- services.html (All services)
- contact.html (Contact form)
- All images (logo + CEO + properties)

## How to Upload:

### GitHub:
1. Upload ALL files to repository root
2. Keep images in same folder as HTML files
3. Enable GitHub Pages
4. Done!

### Hostinger:
1. Upload ALL files to public_html
2. Keep all files together
3. Visit stayza.org
4. Works!

## Important:
- Keep all files in SAME folder
- Don't create subfolders
- Images must be with HTML files
