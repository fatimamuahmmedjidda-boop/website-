## 🚀 Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR-USERNAME/stayza-website.git
   cd stayza-website
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. **Visit**
   - Open `http://localhost:8000` in your browser

## 🌐 Deployment

### Deploy to Vercel

1. Fork or clone this repository
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"
6. Your site is live! 🎉

### Deploy to GitHub Pages

1. Go to your repository **Settings**
2. Navigate to **Pages** section
3. Under "Source", select **main branch**
4. Click **Save**
5. Your site will be live at `https://YOUR-USERNAME.github.io/stayza-website`

### Deploy to Netlify

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your project folder
3. Your site is live! 🎉

## 🔧 Customization

### Update Logo
Replace `./public/10.png` with your logo file

### Update CEO Photo
Replace `./public/Gemini_Generated_Image_v9a1pgv9a1pgv9a1.png` with your photo

### Change Colors
Edit the CSS variables in `index.html` (line ~35):
```css
:root {
    --primary: #03466E;    /* Main blue color */
    --secondary: #EBC487;  /* Gold color */
    --white: #ffffff;      /* White */
}
```

### Update Content
All content can be edited directly in `index.html`:
- Headings
- Paragraphs
- Contact information
- Social media links
